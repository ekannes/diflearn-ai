# DIFLearn AI

Sistem Pembelajaran Berdiferensiasi Berbasis K-Means Clustering untuk membantu guru mengelompokkan siswa berdasarkan hasil belajar dan memberikan rekomendasi pembelajaran yang sesuai.

## Demo Online

🌐 Website:

https://ekannes.pythonanywhere.com

## Fitur

- Upload data rapor Excel
- Analisis K-Means Clustering
- Dashboard visualisasi hasil belajar
- Pengelompokan siswa menjadi Tinggi, Sedang, dan Rendah
- Rekomendasi pembelajaran berdiferensiasi

## Teknologi

- Python
- Flask
- SQLite
- Pandas
- NumPy
- Scikit-Learn
- Chart.js
